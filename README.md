## 📌 Age Eligibility Checker

A simple Age Eligibility Checker web application that determines whether a user is allowed to enter a website based on their age.

Users who are 16 years or older are allowed to access the site, while users below 16 years are restricted.

This project demonstrates the use of JavaScript conditional statements, user input handling, and basic validation logic.

##  📖 Project Overview

The Age Eligibility Checker asks users to enter their age.
After entering the age, the system checks:

If age ≥ 16 → Access Allowed
If age < 16 → Access Denied

The application then displays an appropriate message to inform the user whether they can enter the site or not.

This project is useful for understanding decision-making logic in JavaScript and is suitable for beginners learning programming fundamentals.

⚙️ How It Works
User enters their age in the input field
User clicks the Check Eligibility button
JavaScript checks the age using conditional statements
A message is displayed based on the result:

Example logic:

if (age >= 16) {
    message = "Access Allowed";
} else {
    message = "Access Denied";
}
